## Node.js とパッケージのインストール方法

以下のコマンドで、.tool-versions に書かれたバージョンの Node.js をインストールしてください。

```console
asdf install
```

以下のコマンドで、package.json に書かれたライブラリをインストールしてください。

```console
npm install
```

## .ts ファイルの実行方法

以下のようにして、.ts ファイルを実行することができます。

```console
npx ts-node hello.ts
```
